# Fourier-5.3
## Описание задачи
Для передачи информации по кабелю используется серия импульсов прямоугольной формы, амплитудой 1 В. На втором конце кабеля стоит приемник, который считает сигнал выше 0.8 В единицей, а ниже 0.2 В нулем. Кабельная линии имеет зависимость скорости прохождения отдельных гармоник сигнала от частоты (грубая оценка: v = c/sqrt(f/f0), где f - частота, f0 - некоторая константа, имеющая размерность частоты). Определить максимальную частоту передаваемого сигнала по линии длиной 10 м, 100 м, 1 км.  (два соседних импульса не должны перекрываться.)
